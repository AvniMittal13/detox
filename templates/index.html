About

DETOX is a text analysis tool to help to measure the toxicity in various text and youtube comment section

The technology used includes :

Youtube API

We use the youtube API to fetch comments of a video chosen by the user. The user has to enter the youtube video id to search for the video.

Perspective API

We use google's Perspective API for comment analysis which rates the text according to various measures

Toxicity Measures

We display information based on various toxicity levels which include toxicity, insult, profanity and threat.

Multi-lingual Sentiment Analysis

Data from multiple languages can be analysed, helping people all over the world get an idea about a messgae.
